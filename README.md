# Smart Uttarakhand AI Travel Assistant

## About Project
This is a Python-based terminal application that acts as a smart travel assistant. It suggests tourist places in Uttarakhand based on user preferences and budget.

## Features
- Preference-based suggestions
- Budget filtering
- Detailed destination information
- Interactive menu-driven system

## How it works
1. User selects preference (Adventure/Hill Station/Multiple)
2. System shows matching destinations
3. User enters budget
4. System filters places
5. User selects a place for detailed information
6. Options to explore more or exit
